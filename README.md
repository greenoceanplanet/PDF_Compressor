# PDF 이미지 압축기 (PDF Image Compressor)

이미지가 많이 포함된 무거운 PDF 파일의 용량을 실시간으로 확인하며 효과적으로 줄일 수 있는 데스크톱 애플리케이션입니다.

## 🌟 주요 기능

- **실시간 화질 비교**: 원본(무손실)과 압축 후의 화질을 **실시간 좌우 분할 화면(Side-by-Side)**으로 비교할 수 있습니다.
- **동기화된 미리보기**: 양쪽 화면의 스크롤과 줌이 완벽하게 동기화되어 디테일한 화질 손실 여부를 직관적으로 확인 가능합니다.
- **다양한 보기 모드 지원**:
  - 화면 전체 맞추기
  - 가로 폭 맞추기
  - 세로 높이 맞추기
  - 1:1 실제 크기 (드래그 앤 드롭 패닝 지원)
- **정밀한 압축 설정**: 
  - **DPI 조절 (72 ~ 300)**: 문서 내 이미지의 해상도를 조절합니다.
  - **Quality 조절 (10% ~ 100%)**: JPEG 압축률을 정밀하게 타겟팅합니다.
  - 슬라이더 및 직접 숫자 입력을 모두 지원합니다.
- **예상 용량 계산**: 전체 문서를 압축하기 전에 예상 결과 용량을 미리 계산하여 보여줍니다. (앞부분 샘플링을 통한 높은 정확도)

## 🛠 기술 스택

- **Python 3.12+**
- **PyQt6**: 데스크톱 GUI 환경 구성
- **PyMuPDF (fitz)**: PDF 파싱, 렌더링 및 재구성 코어 엔진

## 🚀 설치 및 실행 방법

1. 저장소를 클론하거나 다운로드합니다.
2. 필요한 Python 패키지를 설치합니다:
   ```bash
   pip install -r requirements.txt
   ```
3. 메인 스크립트를 실행합니다:
   ```bash
   python main.py
   ```

## 📝 사용 방법

1. **[PDF 파일 열기]** 버튼을 클릭하여 용량을 줄일 PDF 파일을 불러옵니다.
2. 좌측 패널에서 **품질(Quality)**과 **DPI** 슬라이더를 조절합니다.
3. 우측 미리보기 패널에서 원본과 압축본의 화질 차이를 실시간으로 비교합니다.
4. 하단 네비게이션(이전/다음 버튼 또는 직접 페이지 번호 입력)을 통해 여러 페이지의 화질을 점검합니다.
5. 계산된 '예상 크기'가 마음에 들면 **[압축 저장하기]** 버튼을 눌러 새로운 PDF 파일로 추출합니다.

## 🔒 보안 및 처리 방식

본 프로그램은 클라우드나 외부 서버로 파일을 전송하지 않습니다. 모든 PDF 렌더링 및 압축 프로세스는 **사용자의 로컬 컴퓨터 메모리 내에서 100% 안전하게 처리**되므로, 기밀 문서나 사내 보안 문서도 유출 걱정 없이 안심하고 압축할 수 있습니다.
